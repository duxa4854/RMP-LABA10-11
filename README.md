# Название проекта

**Название:** MyApp  
**Описание:** Мобильное приложение для учёта личных финансов, позволяющее пользователям отслеживать доходы и расходы, формировать отчёты и планировать бюджет.

## Цель работы

Цель лабораторной работы — отработать навыки разработки мобильного приложения под Android с использованием современных инструментов и подходов:  
* освоить работу с Jetpack Compose для построения UI;  
* реализовать взаимодействие с локальной базой данных (Room);  
* научиться организовывать навигацию между экранами;  
* закрепить принципы модульной архитектуры приложения.

## Функциональность


Приложение поддерживает следующие возможности:  
* регистрация и авторизация пользователя;  
* добавление доходов и расходов с указанием суммы, категории и даты;  
* просмотр списка транзакций с фильтрацией по периоду и категории;  
* формирование отчётов о доходах и расходах за выбранный период;  
* визуализация данных в виде графиков (круговая и столбчатая диаграммы);  
* настройка категорий транзакций;  
* экспорт данных в CSV‑файл;  
* тёмная и светлая темы интерфейса.

## Технологии

В проекте использованы следующие технологии и библиотеки:  
* **Язык программирования:** Kotlin;  
* **Фреймворк для UI:** Jetpack Compose;  
* **Архитектура:** MVVM (Model‑View‑ViewModel);  
* **База данных:** Room (локальное хранение данных);  
* **Навигация:** Jetpack Navigation;  
* **Работа с датой и временем:** Kotlinx.datetime;  
* **Графики:** MPAndroidChart;  
* **Dependency Injection:** Hilt;  
* **Сетевые запросы (опционально):** Retrofit (для синхронизации с сервером);  
* **Дизайн‑система:** Material Design 3.
